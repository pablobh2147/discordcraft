# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-06-25

### Added
- Discord integration plugin/mod for Minecraft servers
- Bidirectional chat relay between Minecraft and Discord
- Player event notifications (join, leave, death, kill)
- Server status announcements (start/stop)
- Discord slash commands for server management (`/ban`, `/pardon`, `/whitelist`, `/stop`, `/playerlist`)
- Channel linking with customizable message-type filters
- Fully configurable messages via `messages.yml`
- Customizable command configuration via `commands.yml`
- Bot activity status customization
- Support for Spigot and Spigot forks (Paper, Purpur, etc.)
- NeoForge mod support for Minecraft 1.20.5+
- Platform-agnostic common core for cross-platform compatibility
- Automated setup command (`/setup`) for easy initial configuration
