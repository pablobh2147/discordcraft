## What's New in v2.0.0

DiscordCraft is a Discord integration plugin/mod for Minecraft servers that bridges your server with Discord, allowing seamless communication and server management.

### Features

- **Bidirectional Chat** — Messages flow between Minecraft and Discord in real-time
- **Player Events** — Join, leave, death, and kill notifications sent to Discord
- **Server Status** — Automatic start/stop announcements
- **Discord Slash Commands** — Manage your server from Discord with `/ban`, `/pardon`, `/whitelist`, `/stop`, `/playerlist`, and more
- **Channel Linking** — Link multiple Discord channels with customizable message filters
- **Full Customization** — Configure all messages and commands via YAML files
- **Bot Activity** — Set custom bot status (Playing, Watching, etc.)
- **Multi-Platform Support** — Available for Spigot/Paper/Purpur (1.13+) and NeoForge (1.20.5+)
- **Easy Setup** — Automated `/setup` command for quick configuration
